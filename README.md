# TinyNvidiaUpdateChecker

#About
Tiny application which checks for NVIDA GPU drivers at each startup, developed in C# for Windows.

#Command line arguments:
`-quiet`: no ui.

#Requirements
+ .NET Framework 4.6
+ Stable internet connection or the application will not run.
